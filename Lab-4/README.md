# Weather Data Visualizer

This project analyzes real-world weather data using Python.  
It loads a CSV file, cleans it, performs statistics, and creates visual charts.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Files Included
- main.ipynb → full Python code
- cleaned_weather.csv → cleaned dataset
- PNG graphs → temperature, rainfall, humidity
- report.md → explanation of results

## How to Run
1. Place your weatherdata.csv file in the same folder.
2. Open main.ipynb in Jupyter or VS Code.
3. Run the code cells.
4. The program will generate:
   - cleaned_weather.csv
   - PNG graph images

## Results
The project shows temperature trends, rainfall totals, and humidity patterns.
